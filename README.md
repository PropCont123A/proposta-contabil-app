## 🎯 Sobre o Sistema

Sistema completo para geração de propostas contábeis com design profissional e navegação simples entre páginas.

## ✅ Características

### **Design Original Preservado**
- ✅ Paleta de cores exata dos arquivos validados
- ✅ Layout idêntico aos designs aprovados
- ✅ CSS Variables organizadas profissionalmente
- ✅ Responsividade mantida

### **Navegação Simples**
- ✅ Páginas HTML separadas (mais confiável)
- ✅ Links diretos entre páginas
- ✅ Menu lateral replicado em todas as páginas
- ✅ Sem JavaScript complexo de SPA

### **Funcionalidades Completas**
- ✅ Sistema de login/logout
- ✅ Dashboard com estatísticas
- ✅ Geração de propostas com múltiplos serviços
- ✅ Gerenciamento de clientes
- ✅ Lista e gestão de propostas
- ✅ Configurações do escritório
- ✅ Dados persistentes no localStorage

```

```

## 🚀 Como Usar

### **1. Acesso Local**
1. Baixe todos os arquivos
2. Abra o arquivo `index.html` no navegador
3. Use as credenciais: `admin` / `123456`

### **2. Upload para GitHub**
1. Crie um repositório no GitHub
2. Faça upload de todos os arquivos
3. Configure GitHub Pages ou Vercel

### **3. Deploy no Vercel**
1. Conecte o repositório GitHub com Vercel
2. Configure o domínio `app.propostacontabil.com.br`
3. Sistema online automaticamente

## 💾 Armazenamento de Dados

### **localStorage**
- `isLoggedIn`: Status de login
- `currentUser`: Usuário atual
- `users`: Lista de usuários cadastrados
- `propostas`: Todas as propostas criadas
- `clientes`: Todos os clientes cadastrados
- `escritorioConfig`: Configurações do escritório
- `escritorioLogo`: Logo do escritório (base64)
